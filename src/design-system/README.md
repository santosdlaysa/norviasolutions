# Norvia Design System

O Design System Norvia transforma a identidade em decisões reutilizáveis para produtos, sites e materiais de comunicação.

## Princípios

- Clareza antes de ornamentação.
- Estados sempre visíveis.
- Densidade com respiro.
- Acessibilidade como requisito.
- Produto diferencia; Norvia unifica.

## Camadas

1. Foundations: cor, tipografia, espaço, raio, sombra, movimento e grid.
2. Components: botão, link, input, select, checkbox, badge, card, alert, modal, table, tabs e tooltip.
3. Patterns: navbar, sidebar, dashboard, landing page, formulário, empty state e error state.
4. Templates: páginas de produto, apresentação, onboarding e operação.

Use `tokens.css` para integrar os fundamentos ao frontend e `tokens.json` para sincronizar com ferramentas de design.

## Assets visuais

- O PDF do Design System contem pranchas visuais de cores, fontes, espacamento, iconografia e componentes.
- `icons.ts` contem os paths vetoriais dos icones base para uso em SVG.
- A familia tipografica definida e Archivo para interface e Spline Sans Mono para dados, labels e codigo.
- A paleta oficial e formada por Ink, Panel, Panel 2, Bone e os acentos Campo, Cafe, Rosa, Lilas, Ceu e Sol.
